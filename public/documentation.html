myFlix API

The myFlix web application provides users with access to information about different movies, directors, and genres. Users can sign up, update their personal information, and create a list of their favorite movies.

API Endpoint Documentation
Endpoint Method Business Logic Query parameters Data format expected in request body Data format for response data
/movies GET Return a list of ALL movies to the user - None (in URL) JSON
/movies/:title GET Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user title None (in URL) JSON
/movies/:genre/:genreName GET Return data about a genre (description) by name/title (e.g., “Thriller”) genre, genreName None (in URL) JSON
/movies/:directors/:directorName GET Return data about a director (bio, birth year, death year) by name directors, directorName None (in URL) JSON
/users POST Allow new users to register - JSON JSON
/users/:id PUT Allow users to update their user info (username) id JSON JSON
/users/:id/:movieTitle POST Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) id, movieTitle None (in URL) Text confirmation message
/users/:id/:movieTitle DELETE Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed id, movieTitle None (in URL) Text confirmation message
/users/:id DELETE Allow existing users to deregister (showing only a text that a user email has been removed) id None (in URL) Text confirmation message
